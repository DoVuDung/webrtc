WebRTC WIthout SIP
--------------------

How to Run : 
Make sure the PeerConnectionj.s is included eithet by using the downloaded one here or by referencing through url 
Install nodejs ( available for both winodws and linux) and run signaller.js
Provide ip of nodejs server in websocket parameter like ws://<ip>:<port>
Run the html page with tomcat web server   

Description :
Will create a video broadcast channel between two peers .
