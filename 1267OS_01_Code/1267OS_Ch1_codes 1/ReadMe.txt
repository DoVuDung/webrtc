getUserMedia 
--------------------------------

How to Run :
Just open the page with chrome borwser


Description :

The JavaScript function getUserMedia ( also known as MediaStream)  is used to allow the webpage to access users media devices 
like Camera, Microphone using browsers native API, without the need of any other third party plugins like flash or silverlight.

For simple demos of these methods download WebRTC-read-only from github .

Code  to access access the webcam and display the local video in a <video/> element.
